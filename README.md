### 三、功能描述
系统分为（身份）：管理员、仓库管理员（系统内有角色管理，可根据需要拓展角色）。

**仓库管理员用户功能：**
1. 登录
2. 库存查询
3. 进货入库
4. 进货单据查询
5. 当前库存查询


**管理员功能：**
1. 登录
2. 用户管理
3. 角色管理
4. 菜单管理
5. 日志操作
6. 供应商管理
7. 客户管理
8. 商品管理
9. 进货入库
10. 进货单据查询
11. 当前库存查询
12. 销售出库
13. 销售单据查询
